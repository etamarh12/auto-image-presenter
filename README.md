# Auto Image Presenter 📸

A professional Python-based automated slideshow application with keyboard navigation and fullscreen support.

## 🚀 Features
- **Automatic Transitions:** Customizable delay between images.
- **Keyboard Controls:** - `Right Arrow`: Next image
  - `Left Arrow`: Previous image
  - `Escape`: Exit fullscreen/Stop
- **Smart Scaling:** Automatically fits images to screen resolution using Pillow.

## 🛠 Tech Stack
- **Language:** Python 3.x
- **GUI Framework:** Tkinter
- **Image Processing:** Pillow (PIL)
